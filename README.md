# AI Incident Contextual Assessment

This example demonstrates how an AI-powered incident management system uses various contextual data points to provide a more accurate and actionable assessment of an anomaly. Instead of just alerting on a simple threshold, the system gathers information about other metrics, recent deployments, business impact, and scheduled tasks to determine the true nature and severity of an incident.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
